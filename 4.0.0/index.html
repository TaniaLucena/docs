ADempiere® 4.0.0 Documentación

La mayor parte de las empresas adquieren distintos software para cubrir las diferentes necesidades de sus organizaciones, pero tarde se dan cuenta de lo complicado que es compartir y unificar la información que cada modulo y organización obtienen.

ADempiere por su condición de ERP ofrece todo lo que una compañía necesita en un único software, ERPyA quiere ofrecer a sus lectores una amplia gama de documentos estructurados para usuarios de alta y baja experiencia en el manejo de Software ERP, en los cuales se explicarán de forma eficiente y detallada cada proceso de este potente sistema, definiendo paso a paso la gestión de cada uno, sus implicaciones contables, administrativas y fiscales y finalmente la emisión de resultados que le permitirá un optimo control de sus operaciones.

La presente versión, cuenta con un entorno práctico y amigable, dashboard dinámico, ventanas con multivistas integradas y búsquedas automáticas. Dicha versión permite la ejecución de procesos y búsquedas al mismo tiempo. Así mismo, contiene otras mejoras que permiten agilizar los procesos realizados por el usuario en su día a día.

Procedimientos Estandarizados por ERPyA

¿Que es Open Project?
¡Hey! tenemos para tí un instructivo experto de nuestra herramienta para gestión de proyectos, con la que logramos de forma ágil y eficiente el feedback con nuestros clientes.

¿Quieres agendar una reunión?
Las reuniones con clientes, es uno de los grandes problemas de las empresas, puesto que en ocasiones resultan improductivas, ERPyA ha modelado un procedimiento para garantizar la armonía, ¡Conocelo!

¿Quieres conocer Slack?
Generalmente cuando contactas al servicio de soporte por teléfono, Help Desk, formulario Web o correo electrónico se vuelve tan protocolar que tiende a ser agotador, queremos contarte la metodología de ERPyA.

¿Cómo reportar tus incidentes?
Nuestro equipo de soporte tiene la responsabilidad de responder oportunamente y dar soluciones efectivas a nuestros clientes, para esto es necesario un reporte detallado sobre el incidente o problema, puedes conocer nuestras recomendaciones aquí.

Novedades

Funcionalidad
- Login sin Rol
- Punto de Venta bajo Nueva Arquitectura
- POS Multi-moneda
- Búsqueda de Productos con Inventario y Precio
- Creación Rápida del Socio del Negocio
- Parámetros de Reportes Personalizados
- Cambio de Formato de Impresión más Sencillo
- Exportación de Datos en Múltiples Formatos
- Ejecución de Procesos y Búsquedas en Segundo Plano

Estética
- UI más Amigable
- Dashboard Dinámico
- Procesos más Intuitivos
- Reportes más Intuitivos
- Ventanas con Multivistas Integradas
- Paginación por Ventana
- Notificación de Procesos Culminados
- Exportación de Registros del Lado del Cliente

Usabilidad
- Más Práctico para Ejecutar Procesos
- POS más Amigable
- Se Guarda la Última Preferencia por Proceso
- Incorporación de Referencias, Acciones y Relaciones como Estándar para Navegar entre Opciones
- Manejo de Notas Integradas con la Ventana

Rendimiento
- Incorporación de Límite en las Consultas (50 Registros Máximo por Página)
- Mejora en Caché del Lado del Cliente: Procesos, Ventanas Smart Browser
- Ventanas de Millones de Registros son Consultadas en Segundos
- Incorporación de Proxy para Distribución de Carga con el Backend y Posible Balanceo
- Implementación de Middleware para Caché y Redimensionado de Imágenes

Seguridad
- Implementación de Token para Acceso de Terceros
- Capa de Seguridad Adicional entre el Cliente Web y el Backend
- El cliente no tiene manera de saber dónde están alojados los datos o el Backend de ADempiere
- Reinicio de Password
- Bitácora de Cambio Integrada a la Ventana
